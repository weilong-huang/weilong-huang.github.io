四、纯量

纯量是最基本的、不可再分的值。以下数据类型都属于 JavaScript 的纯量。

  • 字符串
  • 布尔值
  • 整数
  • 浮点数
  • Null
  • 时间
  • 日期

基本都是用对象的形式来表示

number : 12.3

isSet : true

friend : ~ //~代表null

date : 1976-07-31

t: !!str true //强制转换数据类型
